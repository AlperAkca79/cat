# What is `cof`?
View the contents of your file in your terminal with `cof`.

# How to Use?
First, you need to add **touch** to system path.

### Usage
```
cof <filename>
```

### Example
**Input:**
`cof VERSION`

**Output:**
```
VERSION:
---------------------------------------------
# | cof_2.1.0
```
